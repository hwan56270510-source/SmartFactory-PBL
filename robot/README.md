# Robot Module

This module controls the robotic system and executes automation tasks based on coordinates received from the vision system.

## Functions

* Socket server operation
* Coordinate reception
* Robot motion control
* Pick-and-place execution

## Files

### robot_socket_server_integrated.py

Receives target coordinates from the vision system and manages robot operations.

### wlkata_controller.py

Provides high-level robot control functions.

### mirobot.py

Low-level communication interface for the WLKATA robot arm.
